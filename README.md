# Practica_Versiones_Despliegue
Esta es una practica de despliegue para mostrar conocimientos acerca del control de versiones

## Tecnologías Utilizadas
* HTML5 y CSS3
* Git para control de versiones

## Instalación (Uso local)
1. Clona el repositorio: `git clone [URL del repo]`
2. Abre `index.html` en tu navegador.

## Estructura del Proyecto
* `/css`: Hojas de estilo.
* `index.html`: Página principal.
* `README.md`: Documentación.

## Despliegue
Para desplegar e proyecto solo necesitas clonar este repositorio en una carpeta de HTDOCS

Luego inicar Apache por medio de XAMPP

Por ultimo utilizar la ruta http://localhost/{nombreCarpeta}/Practica_Versiones_Despliegue
